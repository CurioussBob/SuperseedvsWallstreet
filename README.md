# Seed vs Wall Street

Seed vs Wall Street is a tower defense game where players protect their Seed Bank from oncoming enemies using plants that shoot Seed and Ethereum tokens. The game is built on the concept of the Superseed blockchain.

## Game Features
- Plants with different power levels (low, medium, high)
- Enemies that increase in speed as the game progresses
- Resource management system based on Seed tokens
- A dynamic shield bar that fills up as enemies are defeated
- Smooth animations for plants and enemies

## How to Run
1. Clone the repository
2. Install dependencies:
   ```
   pip install -r requirements.txt
   ```
3. Run the game:
   ```
   python src/main.py
   ```
